Real world example
A cinema with rows of seats
Imagine you're an usher checking every seat in a cinema. You start at row A, check seat 1, seat 2, seat 3... When you finish that row, you move to row B and start again from seat 1. That's exactly what two nested loops do.
# outer loop: one step per ROW for row_letter in ["A","B","C","D"]: # inner loop: one step per SEAT in that row for seat_number in [1, 2, 3, 4, 5]: print(row_letter, seat_number)

Understanding the structure
What "nested" actually means
Every time the outer loop takes one step, the inner loop runs all the way through. So with 4 rows and 5 seats: the body runs 4 × 5 = 20 times total.
rows = 4 seats_per_row = 5 for row in range(rows): # runs 4 times for seat in range(seats_per_row): # runs 5 times PER ROW print(row, seat) # runs 4×5 = 20 times total
Note: range(4) gives you 0, 1, 2, 3 — not 1, 2, 3, 4. If you want to start at 1, use range(1, 5).

Scanning a mask pixel by pixel
The mask is just a grid of pixels. To check every pixel, we use two nested loops — outer loop goes row by row (top to bottom), inner loop goes pixel by pixel across each row.
for y in range(image_height): # outer: each row top→bottom for x in range(image_width): # inner: each pixel left→right colour = mask[y, x] if colour == 255: print(x, y, "is white")
